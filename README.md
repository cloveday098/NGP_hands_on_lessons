# **NGP Hands-on Lessons**

This repository is a collection of instructional materials covering topics in basic programming, data science, and quantum with applications in high-performance computing (HPC), artificial intelligence (AI), quantum computing (QC). Each lesson is designed to be self-contained and includes presentation materials, premade scripts or notebooks, instructor notes, and additional resources.

## Repository Structure

Each lesson is organized into its own directory and typically contains:

* **Slides** – Presentation slides used during instruction
* **Scripts / Code Examples** – Demonstration programs, sample datasets, and/or Jupyter notebooks
* **Step-by-Step Instructional README** – An overview of the lesson, screenshots of code or plots, and links to additional resources.

# Intended Audience

These lessons are intended for high school students with both no prior and prior programming experience who are interested in areas such as:

* Scientific Computing
* High-Performance Computing (HPC)
* Artificial Intelligence and Machine Learning
* Data Science
* Quantum Computing
* General Python Programming

# Prerequisites

Individual lessons may have different prerequisites, but students should generally need very little background knowledge to complete these lessons. However, it would be beneficial to have had some exposure to

* Basic Python programming
* Variables, loops, functions, and conditionals
* Installing and importing Python packages
* Running Python scripts or Jupyter notebooks

# Repository Goals

This repository aims to provide instructional materials that emphasize:

* Practical programming skills
* Hands-on coding demonstrations
* Data visualization and analysis
* Computational thinking and problem solving
